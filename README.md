# Quick Reminder #
The One-Click Reminder for Chrome with Notification Center support(Mac OS X 10.8+)

**Feature**

1. **No** Options/Settings
2. **No** Title/Content
3. **No** Cancel
4. **No** Customize
5. **Just one click**
6. with Notification Center support(Mac OS X 10.8+)
